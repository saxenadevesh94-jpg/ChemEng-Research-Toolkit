# ChemEng Research Toolkit

ChemEng Research Toolkit is an open-source starter project for chemical engineering research workflows.

This repository is intentionally simple at the beginning. The goal is to create a clean, beginner-friendly structure for future tools, examples, and experiments.

## Project purpose

This project will help organize:
- experimental data handling
- visualization helpers
- engineering calculations
- machine learning experiments
- reusable utility functions

## Current structure

- data/ - place for datasets and input files
- docs/ - documentation and notes
- examples/ - small usage examples
- src/ - source code modules
- tests/ - test files for future development

## Suggested next steps

1. Add a small sample dataset to data/
2. Create a simple example in examples/
3. Start implementing modules inside src/
4. Add tests as features are added

## Notes

This project currently contains only the initial scaffold. No features have been implemented yet.
